# ხმის ანალიზატორი (Dual Channel Audio DSP Analyzer)

ეს არის ვებ-ტექნოლოგიებზე დაფუძნებული რეალური დროის აუდიო ანალიზატორი, რომელიც შექმნილია გარემო ხმაურის დეტალური ციფრული დამუშავებისთვის (DSP). სისტემა საშუალებას იძლევა მოხდეს ხმის ვიზუალიზაცია და პარამეტრული ანალიზი 11 სხვადასხვა მახასიათებლის მიხედვით.

> [!IMPORTANT]
> ეს პროექტი და მისი პროგრამული კოდი შექმნილია **Gemini**-ს (Google AI) დახმარებით.

## ფუნქციები
*   **11 რეალური დროის გრაფიკი:** ხმის ფიზიკური და ფსიქოაკუსტიკური მახასიათებლების მონიტორინგი.
*   **ფიქსირებული მასშტაბი (Fixed Scale):** გრაფიკები ინარჩუნებენ მუდმივ მასშტაბს, რაც მონაცემთა ობიექტური შედარების საშუალებას იძლევა.
*   **ორმაგი წყარო:** ანალიზი პირდაპირ მიკროფონიდან ან ატვირთული აუდიო ფაილიდან.
*   **სტერეო მონიტორინგი:** დამოუკიდებელი ანალიზი მარცხენა (L - მწვანე) და მარჯვენა (R - წითელი) არხებისთვის.
*   **ლოკალიზაცია:** ინტერფეისი სრულად ქართულ ენაზეა.

## ტექნოლოგიური სტეკი
*   **Meyda.js** — აუდიო მახასიათებლების ექსტრაქცია (Feature Extraction).
*   **Chart.js** — მონაცემთა დინამიკური ვიზუალიზაცია.
*   **Web Audio API** — აუდიო ნაკადების მართვა ბრაუზერში.
*   **Vanilla JS, HTML5, CSS3.**

## გაშვების ინსტრუქცია
ბრაუზერის უსაფრთხოების წესების გამო, მიკროფონთან წვდომისთვის აუცილებელია პროექტის გაშვება ვებ-სერვერის მეშვეობით:

1.  გადმოწერეთ პროექტის ფაილები.
2.  გახსენით **VS Code**-ში და გამოიყენეთ **Live Server** გაფართოება.
3.  ან გამოიყენეთ ტერმინალიდან: `python -m http.server 8000`.
4.  ბრაუზერში გახსენით მისამართი: `http://localhost:8000`.

## DSP პარამეტრების განმარტება

| პარამეტრი | აღწერა | მასშტაბი |
| :--- | :--- | :--- |
| **RMS** | ხმის საშუალო სიმძლავრე (მოცულობა). | 0 - 0.5 |
| **Energy** | სიგნალის ჯამური ენერგია. | 0 - 100 |
| **ZCR** | Zero Crossing Rate - ხმის "ხორკლიანობა". | 0 - 300 |
| **Spectral Centroid** | ხმის "სიკაშკაშე" (სპექტრის ცენტრი). | 0 - 11kHz |
| **Spectral Flatness** | გვიჩვენებს, ხმა ტონალურია თუ ხმაური. | 0 - 1.0 |
| **Spectral Rolloff** | სიხშირე, სადაც ენერგიის 90% იყრის თავს. | 0 - 11kHz |
| **Spatial Pan** | ხმის გადახრა მარცხნივ ან მარჯვნივ. | -1 - +1 |
